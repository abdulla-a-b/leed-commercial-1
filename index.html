How to read this

Every LEED prerequisite and credit is assigned to one accountable department. Each task is tagged Achieve (one-time, to win certification) or Maintain (recurring, to keep it). Department completion is weighted: a finished task counts full, an in-progress task counts half.

The business case

Bangladesh hosts 270 LEED-certified apparel factories — more top-rated green factories than any other country. Certification is no longer a badge: green-certified plants attract long-term sourcing commitments, stronger price discussions, and access to sustainability-linked loans and concessional green credit. For yarn dyeing, the water-recycling and ETP investments that earn Water Efficiency points also cut freshwater intake and effluent-treatment cost, so the credit chase and the cost-down agenda point the same direction.

Connect the backend

Wire to Google Sheets in 2 steps. 1) Deploy apps-script/Code.gs as a Web App (Execute as: Me · Access: Anyone). 2) Paste the Web-App URL into CONFIG.API at the top of the page script. Until then the form runs in demo mode and updates the in-memory dashboard only.
The Python pipeline in pipeline/leed_pipeline.py reads the same sheet nightly via GitHub Actions, recomputes weighted readiness & threshold gaps, and writes data/leed_state.json back to the repo — this dashboard then loads from that file on Pages.
